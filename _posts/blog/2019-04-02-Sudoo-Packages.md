## 介绍

NodeJS 自带的库还可以吧。。但是贼难用，乏善可陈。比如说 HTTP 和 HTTPS 请求要用不同的包，比如说一些基础函数的缺失，比如说 `Object.keys` 居然能那么慢。

以下的轮子都是及其切合我实际使用的。我很难吹嘘它们有多么有技术含量，不过 API 的设计我都是用了心的，合理，干净，语意化，无副作用，覆盖全面。 PR 或者 ISSUE 都欢迎。
