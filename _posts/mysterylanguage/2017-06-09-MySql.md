# 将 .sql 文件导入某个库

```bash
importdb
mysql -uroot -proot
>DROP DATABASE feedlot; CREATE DATABASE feedlot; exit;
mysql -uroot -proot feedlot < database.sql 
```
<del>这个还可以</del>

# 更新 （2017-12-27）

本质上 `.sql` 文件就是一些sql命令的集合，所以只要将每一个命令输入数据库就可以将数据库恢复到存贮的状态。
